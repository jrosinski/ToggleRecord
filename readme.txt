Log in to the codec webUI, go to Integration, In-Room Control.
Launch Editor.
Import the roomcontrolconfig.xml under Import Configuration from File.
Export to Video System.

Open Macro Editor under Integration.
Click Import from File and upload the ToggleRecord.js
Modify the START_RECORDING and STOP_RECORDING variables at the top of the macro to reflect the correct DTMF tones for your environment.
Click Save and toggle the macro on.

Recording Panel will only show while in a call.
